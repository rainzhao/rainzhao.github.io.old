1.一个简单的指令

（1）指令的四种使用方式 //a.html <hello></hello> <div hello></div> <div class="hello"></div> <!-- directive:hello --> //a.js var myModule = angular.module('helloDirective',[]); myModule.directive('hello',function(){ return{ restrict: 'ECAM', template: '<div>Hi angular!</div>', replace: true } }); （2）使用templateUrl加载指令 这样加载可写更多的html代码 myModule.directive('hello',function(){ return{ restrict: 'ECAM', templateUrl: 'templates/hello.html', replace: true } }) （3）使用$templateCache将模版缓存起来然后加载
